Add more styles to the WordPress Dashboard - Añade más estilos al escritorio de WordPress

== Description ==

Añade esquemas de colores adicionales a WordPress, haciendo que sea realmente personalizable el escritorio de administración.

== Installation ==

1. Descomprimir el archivo en algún lugar de su PC
2. Subir la carpeta 'stylepress' a tu WordPress
3. Activar el plugin a través del menú 'Plugins' de WordPress.
4. Ir a 'Usuarios &raquo; Tu perfil' y selecciona el esquema de colores que más te agrade.

== Frequently Asked Questions ==

= Español =

= ¿Que hace StylePress? =

Añade esquemas de colores adicionales. Estos esquemas son solo visibles para usuarios registrados y solo modifican los colores y algunas imagenes de los estilos originales, dandole un aspecto más renovado al panel de administración de WordPress.

= ¿Debo editar algún archivo para que funcione correctamente? =

No. StylePress funciona sin que debas tocar código. Lo que si debes hacer es seleccionar, desde tu perfil, el esquema de colores que deseas utilizar ya que, de lo contratrio, es muy probable que sigas viendo tu panel de administración sin ningún cambio a pesar de haber activado el plugin. 

= Quiero añadir mi propio estilo personalizado ¿Como lo hago? =

Esperamos que para el lanzamiento de la versión 1.0, tengamos disponible en la web oficial una hoja de estilos en blanco, para que cualquier usuario entusiasta cree su propio estilo personalizado y, si lo desea, lo suba y comparta con el mundo.

Más información sobre StylePress en la página oficial [El Otro Yo &raquo; Proyectos &raquo; StylePress](http://elotroyo.co.cc/proyectos/stylepress)
